# 课件模板

本文档提供计算材料学课程的幻灯片结构模板，供智能体生成课件时参考。

---

## 课件结构规范

### 整体结构

```
幻灯片1：封面
幻灯片2：目录
幻灯片3-N：正文内容
幻灯片N+1：总结
幻灯片N+2：习题/思考题
幻灯片N+3：参考文献
幻灯片N+4：致谢
```

### 设计原则

1. **每页信息量**：控制在3-5个要点，避免信息过载
2. **字体大小**：标题≥32pt，正文≥24pt，确保后排可见
3. **配色方案**：主色调统一，背景与文字对比度高
4. **图表质量**：清晰美观，标注完整

## 图表设计规范

### 图表类型选择

| 内容类型 | 推荐图表 | 示例 |
|---------|---------|------|
| 流程 | 流程图 | 计算步骤 |
| 对比 | 表格、柱状图 | 方法对比 |
| 趋势 | 折线图 | 能量收敛 |
| 分布 | 热图、等高线 | 电荷密度 |
| 结构 | 结构图、模型图 | 晶体结构 |
| 关系 | 韦恩图、关系图 | 方法关系 |

### 图表制作要求

1. **清晰度**：分辨率不低于150dpi
2. **标注**：坐标轴标签、图例完整
3. **配色**：与课件风格一致
4. **引用**：标注来源（如非原创）

## 动画设计建议

### 动画类型

1. **出现动画**：文字、图片依次出现
2. **强调动画**：重点内容闪烁或放大
3. **过程动画**：展示计算过程、物理过程
4. **转场动画**：页面切换效果

### 使用原则

- 适度使用，避免花哨
- 突出重点内容
- 控制动画速度
- 保证兼容性

## 课件生成流程

### 智能体生成课件时的步骤

1. **确定主题**：明确课件的核心内容
2. **结构规划**：按模板设计幻灯片顺序
3. **内容填充**：填充文字、公式、要点
4. **图表设计**：描述需要的图表类型和内容
5. **细节完善**：检查完整性、逻辑性

### 输出格式

智能体生成课件时，应输出：

```markdown
# 课件标题

## 幻灯片1：封面
- 课程名称
- 章节标题
- 教师信息

## 幻灯片2：目录
- 目录项1
- 目录项2
......

## 幻灯片3：XXX
### 标题
- 要点1
- 要点2
- [图表说明：XXXX图，展示XXXXX]
......
```

这样用户可以将内容复制到PPT软件中快速制作幻灯片。
